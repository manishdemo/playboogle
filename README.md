# Boogle Game

This is a Boogle game developed using React and Ruby on Rails.


#### Run it locally:
* Start your rails server: `rails s`
* Navigate to http://localhost:3000 in your browser.

* Ruby version 2.7.0
* Rail version 6.0.3.1
